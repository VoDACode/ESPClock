# ESPClock

ESPClock is an advanced clock firmware for ESP8266/ESP32 microcontrollers, featuring WiFi connectivity, NTP synchronization, weather data integration, and a customizable status bar. It leverages the HAIOT framework for seamless IoT integration.

## Features

- WiFi connectivity with easy configuration
- NTP time synchronization
- Weather data retrieval from BME280 sensor
- Customizable status bar with various elements
- MQTT support for IoT integration
